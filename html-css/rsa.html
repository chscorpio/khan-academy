Understanding RSA Encryption Using Simple Math

Welcome to this blog post where we explain RSA encryption using simple math concepts, perfect for a 5th grader! RSA is a way to encode (hide) and decode (unhide) messages using numbers. Let’s dive in!

Step 1: Pick Two Prime Numbers

A prime number is a number greater than 1 that can only be divided by 1 and itself. For example, 2, 3, 5, 7, and 11 are prime numbers.

Step 2: Multiply the Primes

Let’s call the two prime numbers p and q. Multiply them together to get a number we call n.

For example:

  • If p = 3 and q = 5, then n = 3 × 5 = 15.
  • If p = 7 and q = 11, then n = 7 × 11 = 77.

Step 3: Find Another Number Called e

We need a number e that is smaller than (p - 1) × (q - 1) and doesn’t share any common factors with it (except 1). This might sound tricky, but it’s just picking a number that “works.”

For simplicity, let’s use e = 3.

Step 4: Find a Special Number Called d

Next, we calculate d, which works like a secret key. It satisfies this equation:

(e × d) % ((p - 1) × (q - 1)) = 1

For example:

  • If p = 3 and q = 5, then (p - 1) × (q - 1) = 2 × 4 = 8. Here, d = 7 works because (3 × 7) % 8 = 1.
  • If p = 7 and q = 11, then (p - 1) × (q - 1) = 6 × 10 = 60. Here, d = 7 works because (3 × 7) % 60 = 1.

Step 5: Encode a Message

Messages are turned into numbers using their ASCII values (every letter has a number). For example, the letter A is 65, B is 66, and so on.

To encode a message, use this formula:

Encoded = (Messagee) % n

Example 1: Encode M (ASCII value 77) using p = 3, q = 5, and e = 3:

Encoded = (773) % 15 = 13

Example 2: Encode M (ASCII value 77) using p = 7, q = 11, and e = 3:

Encoded = (773) % 77 = 36

Step 6: Decode the Message

To decode the message, use this formula:

Decoded = (Encodedd) % n

Example 1: Decode 13 using p = 3, q = 5, and d = 7:

Decoded = (137) % 15 = 77, which is the ASCII value for M.

Example 2: Decode 36 using p = 7, q = 11, and d = 7:

Decoded = (367) % 77 = 77, which is the ASCII value for M.
